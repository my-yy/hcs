如何完成复习

当笔记需要复习时会出现"评分面板"，通过点击评分面板中的按钮完成一次复习：

加入拾忆笔记的内容一般会在“第二天”安排第一次复习，即“今天”加入的内容需要等到“明天”才会显示评分面板。

复习模式

拾忆笔记内置了四种复习算法：

复习算法 简介 适用对象
预设 按自己预先设定的复习间隔进行复习 大体量材料
神经网络 由神经网络算法测算下次复习日期 大体量材料
Anki 基于Anki软件所采用的复习算法 小体量材料
SuperMemo SuperMemo软件曾使用的SM2算法 小体量材料

针对复习材料选择复习模式，复习材料可以分为两大类：

  • 大体量材料：由多个知识点构成，复习一次花费时间较长，例如试题、笔记

  • 小体量材料：知识点单一，复习一次花费时间较少，例如英文单词、单个方程式

大体量材料推荐采用 “预设” 或 “神经网络”复习算法，因为大体量记忆材料不应追求记忆效果

这一观点可能有些 “反常识”，详细论述可参考：《【简明教程】复习算法》 《复习原理》

小体量材料推荐采用Anki或SuperMemo复习算法，并且由于拾忆笔记并非为小体量记忆材料所设计，因此对于小体量材料，专业的SuperMemo软件可能会是更好的选择，并且若小体量材料专指单词的话，更为推荐墨墨记单词等专业词汇类应用。

笔记本

笔记本的复习模式选项仅作为新建笔记时的默认值，与已创建笔记之间并不联动；例如某笔记本复习模式为“预设”，将其调整为“神经网络”后，并不会影响笔记本内现有笔记的复习模式。

印象

将印象笔记添加到拾忆笔记仅是创建一个内容为笔记链接（笔记网址）的复习笔记，如：

https://app.yinxiang.com/shard/s57/nl/2147483647/9e0d54cd-122b-409f-b02e-171c5fb8ee86/

该笔记并无特别之处（你可以手动粘贴该网址或变动该网址为其它内容）
