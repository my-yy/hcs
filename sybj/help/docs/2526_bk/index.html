跑路公告

省流：新版本已经发布；软件计划在2024年8月20日后下线。

这个APP已经近一年没有更新了，遇到了很多变故，当然，深层原因是对软件开发失去了热忱，所以到现在才发布了新版本（1.0.73）：

https://huacishu.com/sybj/

这个版本的特点是解除了所有付费功能限制（未到期的年费用户可以联系我提供退款；此外如果之前付费觉得亏了的也可以退~）

未来会提供更完善的数据导出功能；不过由于不想跟苹果客服打交道，所以不再提供新的iOS版本。

我是从2014年的那个时候开始着手做这个APP的，定到三年后的2024，是凑个10年吧。如今回头再看，折腾这么久，主要原因是方法没找对。有策略上的不足，也有技术选型上的失误，应该“先这样、然后再这样、再接着这样…“，终归是自己的水平太凹了… (￣▽￣)"

不过也不是很悲伤，在K12禁令之前已经靠“遗忘算法API”把投入一股脑赚了回来…

不打算继续投入下去的主要原因在于，一年多前找到了一种新的学习方法，之后没有再继续用过它，如今感觉这类基于遗忘曲线的软件已经意义不大了。

此外，如今还面临新的人生症结，有仇要报，也有东西要找回来，没有过多精力继续投入了，因此，它就这样完结了。
