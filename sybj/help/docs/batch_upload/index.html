批量导入

在“笔记本”中点击“批量导入内容”

拾忆笔记支持导入.xlsx格式的Excel文件，文件第一行应包含 name、content、content2 三个字段，例如：

点击此处下载示例文件

字段说明

name为纯文本字段，不支持图片、链接等语法；content、content2为富文本字段，支持各类语法规则

字段含义受“笔记格式”影响，例如:

对于“笔记”格式笔记：

name为笔记标题，content为笔记内容，忽略content2字段

对于“单页“格式笔记：

content为内容字段，name、content2均被忽略
